# Propozycje poprawek do uchwały URK o sztucznej inteligencji w kształceniu

Platforma konsultacyjna inicjatywy studenckiej do projektu nowelizacji uchwały Uniwersyteckiej Rady ds. Kształcenia UW.

**Autor**: Aleksander Rej, student II roku politologii Uniwersytetu Warszawskiego

**Strona**: https://arejgh.github.io/propozycje-urk-uw/

**Kanał konsultacyjny**: formularz konsultacyjny URK

Materiał konsultacyjny: 13 propozycji poprawek redakcyjnych + 3 wątki analityczne, osadzone w ośmiu autorskich ramach redakcyjnych. Nie stanowi oficjalnego stanowiska żadnego organu Uniwersytetu Warszawskiego.
